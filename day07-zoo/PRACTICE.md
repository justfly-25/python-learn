## 练习 1：添加新动物

仿照 Cat、Dog、Bird 的模式，添加一个 `Fish` 类：
- 属性：name、age、water_type（淡水/海水）
- speak() 返回 "咕噜咕噜~ 🐟"
- 特有方法：swim() 返回游泳描述

```python
fish = Fish("泡泡", 1, "淡水")
print(fish.swim())  # 泡泡 在水里游来游去~
```

## 练习 3：动物园主程序

写一个交互式菜单，让用户可以：
1. 添加新动物（选择种类：猫/狗/鸟 + 输入名字年龄）
2. 查看所有动物信息
3. 让所有动物叫一遍（多态实践）
4. 退出

```python
zoo = []  # 存放所有动物
while True:

```
